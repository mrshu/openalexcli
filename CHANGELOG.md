# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- GitHub Actions workflow for automatic PyPI publishing

## [0.1.5] - 2025-01-23

### Changed

- Simplified tagline to "A CLI for OpenAlex API - designed for humans and AI agents"

## [0.1.4] - 2025-01-23

### Changed

- Clarified BibTeX output format description in README

## [0.1.3] - 2025-01-23

### Changed

- Updated description to highlight support for humans and AI agents

## [0.1.2] - 2025-01-23

### Added

- uv and uvx installation instructions in README

## [0.1.1] - 2025-01-23

### Changed

- Updated README with PyPI installation instructions

## [0.1.0] - 2025-01-23

### Added

- Initial release
- Search for works with filters (date, citations, open access)
- Get work details by OpenAlex ID or DOI
- Export works as BibTeX
- Get citations and references for works
- Author commands (get, search, works)
- Institution commands (get, search, works)
- Source/journal commands (get, search, works)
- Group by functionality for aggregated statistics
- JSON output for piped commands
- Polite pool support via email
